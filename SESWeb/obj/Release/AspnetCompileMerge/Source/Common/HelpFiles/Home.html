The following screen shot is a sample for the Default Page you will see when you login into the system.

The Default page contains following elements

  • Menus

    This helps you to navigate to the page you want to accSESWeb.

  • Page Name

    This indicates the current page you are working on.

  • Back to portal Menu

    This menu will navigate you the Portal where you logged in using your User Name and Password.

  • Birth day list

    This shows you the list of employees celebrating birthday today .

  • Separted list

    This shows you the list of employees leaving from our company today .
